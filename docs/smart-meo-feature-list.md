# Smart MEO 機能一覧（仕様書化ドラフト）

## 1. 順位計測・分析機能

### 1-1. 順位チャート
- キーワードごとの検索順位を可視化
- 時系列での推移をグラフ表示

### 1-2. キーワード分析
- 登録キーワードごとの順位・変動を分析
- MEO対策キーワードの効果測定

### 1-3. カレンダー順位レポート
- 日別で順位履歴を確認
- イベントや施策との相関確認が可能

### 1-4. 日次順位レポート（キャプチャ）
- 実際の検索結果画面を保存
- エビデンスとして利用可能

### 1-5. 多地点順位チェック
- 複数エリアからの検索順位取得
- ローカルSEO対策の精度向上

### 1-6. 順位変動チェック
- 順位の上下変動をトラッキング

### 1-7. 順位変動アラート
- 大きな順位変動時に通知

### 1-8. 施策メモ
- 実施した施策を記録
- 順位変動との因果関係を管理
